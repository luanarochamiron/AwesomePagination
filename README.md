# Awesome Pagination Design

Um componente de paginação (Pagination Bar) moderno, elegante e totalmente responsivo desenvolvido exclusivamente com HTML5 e CSS3. O projeto apresenta uma barra de índices compacta estruturada em blocos contínuos e integrados, ideal para ser acoplada à base de tabelas de dados, listagens de e-commerce, blogs ou painéis administrativos corporativos.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do card principal, cabeçalhos de texto descritivo e blocos sequenciais do formulário de paginação.
* CSS3: Estilização completa, uso de cantos suavizados, gerenciamento de bordas divisórias finas e aplicação de fundos em gradiente.
* Flexbox: Alinhamento horizontal em fluxo contínuo para os blocos numéricos e centralização absoluta de todo o componente na tela.

## Funcionalidades e Técnicas Aplicadas

* Estrutura em Bloco Único Integrado: Ao contrário dos seletores isolados tradicionais, os botões de controle ("Previous" e "Next") e as páginas numéricas (1 a 10) são unidos lado a lado por meio de divisores internos sutis, formando uma barra sólida unificada.
* Estado Ativo com Transparência Suave: A página ativa selecionada (bloco número "5") ganha destaque imediato através de uma variação de tonalidade mais clara e translúcida, indicando intuitivamente a posição do utilizador.
* Identidade Cromática Vibrante: Uso de uma paleta de cores quentes baseada em um tom vermelho-alaranjado sólido para a barra de navegação, contrastando perfeitamente sobre um fundo de página com gradiente linear em degradê laranja.
* Hierarquia Tipográfica Limpa: Inclusão de uma seção de cabeçalho no card branco com um título robusto ("Pagination Design") e um subtítulo descritivo sutil para contextualizar o módulo.
* Elevação Tridimensional Discreta: O contêiner retangular branco utiliza `border-radius` nos cantos e uma leve projeção de sombra (`box-shadow`) na base inferior para criar profundidade em relação ao fundo.
